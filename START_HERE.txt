БЫСТРЫЙ СТАРТ (3 МИНУТЫ):
─────────────────────────────────────────────────────────────────

1️⃣  ЛОКАЛЬНЫЙ ТЕСТ (опционально):

    cd /root/frost-ai/users/583297536/dashboard
    python app.py
    
    Откроешь: http://localhost:8050
    Нажми Ctrl+C для выхода


2️⃣  СОЗДАЙ РЕПОЗИТОРИЙ НА GITHUB:

    git init
    git add .
    git commit -m "Initial: Aidar Dashboard"
    git remote add origin https://github.com/YOUR/aidar-dashboard.git
    git push -u origin main


3️⃣  РАЗВЕРНИСЬ НА RENDER (5 минут):

    1. Перейди: https://render.com
    2. "New Web Service"
    3. Выбери репозиторий
    4. Name: aidar-dashboard
    5. Build: pip install -r requirements.txt
    6. Start: gunicorn --workers 1 --worker-class sync --bind 0.0.0.0:$PORT app:server
    7. "Create"
    
    ⏳ Ждёшь 2 минуты... готово! 🚀
    
    Твой дашборд: https://aidar-dashboard.onrender.com


4️⃣  АВТООБНОВЛЕНИЕ ДАННЫХ (опционально):

    На Render: "New Cron Job"
    Schedule: 0 6 * * *
    Command: python data_updater.py
    
    ✅ Теперь данные обновляются каждый день в 06:00 UTC
