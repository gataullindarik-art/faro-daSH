# 📊 Aidar Business Dashboard

Полнофункциональный дашборд для мониторинга бизнеса на WB, МойСклад и оптовой продажи.

## 🎯 Функциональность

### 📈 **WB Финансы**
- Выручка 60 дней, среднемесячная
- Маржа и прибыль
- Тренд выручки по дням
- Топ-5 товаров по выручке и марже
- Финансовая сводка

### 📦 **Остатки (МойСклад)**
- Распределение по 5 складам
- Топ товаров по остаткам
- Сводка по всем складам

### 💼 **Оптовая продажа**
- Целевая модель по сезонам
- Оптовые цены на 10 топ товаров
- Бюджет май: распределение 1M ₽
- Ожидаемый результат (ROI 90%)

### 📊 **Аналитика**
- Тренды выручки
- Прогноз на 30 дней
- Ключевые метрики (концентрация, оборот)

## 🚀 Установка и запуск

### 1. Установка зависимостей

```bash
cd /root/frost-ai/users/583297536/dashboard
pip install -r requirements.txt
```

### 2. Локальный запуск (для тестирования)

```bash
python app.py
```

Дашборд будет доступен по адресу: **http://localhost:8050**

### 3. Ежедневное обновление данных

Добавь в crontab:

```bash
crontab -e
```

Добавь строку (обновление каждый день в 06:00 UTC):

```bash
0 6 * * * /usr/bin/python3 /root/frost-ai/users/583297536/dashboard/data_updater.py >> /root/frost-ai/users/583297536/dashboard/logs/update.log 2>&1
```

Создай папку логов:

```bash
mkdir -p /root/frost-ai/users/583297536/dashboard/logs
```

## 🌍 Развертывание на облаке (Render, Heroku)

### Для Render.com:

1. Создай файл `render.yaml`:

```yaml
services:
  - type: web
    name: aidar-dashboard
    env: python
    buildCommand: pip install -r requirements.txt
    startCommand: python app.py
    envVars:
      - key: PYTHON_VERSION
        value: 3.11
      - key: PORT
        value: 8050
```

2. Подключи к GitHub репозиторию
3. Деплой будет автоматическим

### Для ежедневного обновления на облаке:

Используй **GitHub Actions** или **Render Cron Jobs** для запуска `data_updater.py` каждый день в 06:00.
