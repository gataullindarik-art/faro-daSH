# 🔄 ОБНОВЛЕНИЕ: Подключение реальных данных с WB API

## ✅ Что изменилось

Дашборд теперь **автоматически подтягивает данные с WB API**!

- ✅ Реальные продажи за последние 30 дней
- ✅ Актуальное количество заказов
- ✅ Дневные графики выручки
- ✅ Реальные товары и маржи
- ✅ Автообновление каждый день в 06:00 UTC

---

## 🚀 Как это работает

### **1. Скрипт обновления** (`data_updater.py`)

Запускается каждый день в 06:00 UTC и:

1. Загружает **продажи с WB API** (за 30 дней)
2. Загружает **заказы с WB API** (за 30 дней)
3. Загружает **остатки с МойСклад API**
4. Обрабатывает данные
5. Сохраняет в JSON файлы (`data/wb_report.json`, etc.)

### **2. Дашборд читает эти JSON файлы**

При открытии дашборда он:
- Загружает актуальные данные из JSON
- Если файлы есть → показывает реальные цифры
- Если файлов нет → показывает тестовые данные для демо

## 💡 Если что-то не работает

### WB API ошибка 429 (лимит запросов)

Скрипт автоматически ждёт 60 сек и повторяет запрос.
Это нормально, просто WB ограничивает частоту запросов.

### Файлы wb_report.json пустые

Проверь:
1. Токен WB в правильном месте
2. Токен имеет права на `api/v1/supplier/sales` и `api/v1/supplier/orders`
3. Есть ли у тебя продажи за последние 30 дней

### МойСклад не обновляется

Проверь:
1. Токен МойСклад в правильном месте
2. Токен имеет права на `entity/product`
